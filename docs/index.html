Capabilities

Everything you need,
nothing you don't

Pure Go. No database required. No cloud dependency. Just a single binary that turns your local machine into a personal AI knowledge hub.

Knowledge Base

Store and retrieve information using natural language. Supports text, images (with vision), and PDF documents. Auto-generated keywords for intelligent search.

Smart Reminders

Natural language scheduling: one-time, daily recurring, or specific date/time. Delivered across terminal, WeChat, and desktop interfaces.

Multi-Interface

Terminal REPL for power users, native desktop app via Wails, and WeChat bridge for mobile access. Same brain, every surface.

AI-Powered Chat

Powered by OpenAI and Anthropic models with hot-swappable profiles. Intent recognition routes your queries to the right handler automatically.

Skills System

Extensible prompt-based skill packs loaded on demand. Influence routing, translation, search behavior, and response style.

Local-First & Private

All data stays on your machine. JSON file storage, encrypted API keys, no telemetry. Your memories belong to you.

Conversation Modes

Three ways to think

Switch between modes to control how myclaw processes your queries — from direct AI chat to knowledge-grounded reasoning to autonomous tool use.

direct

Direct Mode

Questions go straight to the AI model without consulting the knowledge base. Best for general conversation, brainstorming, and quick questions that don't need your personal context.

knowledge

Knowledge Mode

Every query is enriched with relevant entries from your local knowledge base. The AI reviews and synthesizes your stored information before responding. Perfect for recall and research.

agent

Agent Mode

Full tool-calling capability: search the web, read/write memory, manage reminders, and integrate with external services via MCP/NCP/ACP protocols. The AI acts, not just answers.

Get Started

Up and running in seconds

Clone, build, run. No complex setup, no external services required.

terminal
# Clone the repository
git clone https://github.com/xzg/myclaw.git
cd myclaw

# Build for your platform
make build-current

# Run the terminal REPL
./build/myclaw

# Or build the desktop app (requires Wails)
make build-desktop

# Cross-platform builds
make build-windows # Windows + NSIS installer
make build-macos # macOS + DMG
make build-linux # Linux
usage
# Remember something
/remember The WiFi password for the office is quantum42fish

# Ask from your knowledge base
What's the office WiFi password?

# Set a reminder
/notice 每天 09:00 Stand-up meeting

# Manage knowledge
/list # List all entries
/forget # Remove an entry
/append # Add to an existing entry

Architecture

Clean & modular

Intentionally simple. Pure Go with zero database dependencies. Every component has a clear boundary and single responsibility.

internal/ai/
OpenAI & Anthropic integration, intent routing, search planning
internal/knowledge/
Local knowledge base: JSON storage, keyword search, entry management
internal/reminder/
Cron-like scheduling, multi-channel delivery, natural language parsing
internal/app/
Core service layer, conversation modes, tool provider orchestration
internal/weixin/
WeChat bridge: QR login, message polling, text & voice handling
internal/terminal/
Interactive REPL for CLI usage with rich command support
internal/skilllib/
Skill/prompt pack system with hot-loading and routing influence
internal/fileingest/
File processing pipeline: image vision, PDF text extraction
